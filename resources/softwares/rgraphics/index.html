Gallery for R graphs and charts

Sunday, 11 Aug 2019 Softwares, Data Visualization, Programming, Tutorials

R is a great tool for data visualization. The native tool becomes even better thanks to external packages such as ggplot2 that are widely used in the gallery. If you’re wondering what you can do with R in terms of dataviz, the following online resources may help you.

From Data to Viz is a classification of chart types based on input data format. It comes in the form of a decision tree leading to a set of potentially appropriate visualizations to represent the dataset. The project is built on two underlying philosophies. First, that most data analysis can be summarized in about twenty different dataset formats. Second, that both data and context determine the appropriate chart. Thus, From Data to Viz suggested method consists in identifying and trying all feasible chart types to find out which suits your data and idea best. Once this set of graphics is identified, From Data to Viz aims to guide you toward the best decision. It also provides a list of common caveats to avoid and always provides a reproducible code snippet in the R programming language. Dataviz is a world with endless possibilities and this project does not claim to be exhaustive. However, it should provide you with a good starting point.

The R CHARTS web site has been created without any financial support by José Carlos Soage. It was generated with blogdown, an R package which allows creating static web sites using the HUGO generator. The theme of this site is a custom theme inspired by Kross and Academic themes. The classification of the charts is based on the Financial Times visual vocabulary. On the site of R CHARTS you will find code examples of R graphs made with base R graphics, ggplot2 and other R package.

R graph gallery is a collection of charts made with the R programming language. Charts are organized by family and types. Hundreds (~ 400 charts) of charts are displayed in several sections, always with their reproducible code and explanations available. The gallery makes a focus on the tidyverse and ggplot2. For each chart, a focus is made on code, but a link toward data-to-viz is provided to give more information concerning the theory.
